# Megha Bhasker — Portfolio

Personal portfolio site. Built with plain HTML/CSS/JS + Vite.

## Local development

```bash
npm install
npm run dev
```

Opens at http://localhost:5173

## Add your resume PDF

Drop your resume PDF into the root folder and name it:
```
megha_bhasker_resume.pdf
```

The download button in the Contact section will automatically pick it up.

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to vercel.com → Add New Project → import the repo
3. Vercel auto-detects the config — just click Deploy

Your site will be live at `your-project.vercel.app` in ~1 minute.

Every `git push` auto-redeploys.

## Project structure

```
megha-portfolio/
├── index.html        ← all content lives here
├── style.css         ← all styles
├── main.js           ← scroll animations, nav behavior
├── package.json
├── vite.config.js
├── vercel.json
└── megha_bhasker_resume.pdf   ← add this yourself
```
